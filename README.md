**При создании сайта использовались технологии:**

* Файловая структура Nested

* Flex-box

* Адаптивность 320px - 1280px+
* Медиа запросы
* Оптимизация изоброжений
* Подключение шрифтов через font-face
* Шрифт Inter (400, 700, 900)
